## basic 

### how the components communicate 

```sh

props - 实现父子/子父/兄弟组件通信

customized event - 实现子父组件通信

global event bus - 任意组件通信

pub-sub 模式 - 任意组件通信 (用的少, 一般用于 小程序...)

vuex - 集中式状态管理器, 实现任意通信 (vue3 推荐 pinia)

ref - 父组件组件获取子组件实例

slot - 插槽, 实现父子通信

```
