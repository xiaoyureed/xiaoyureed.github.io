# aws

## 提供的服务

```

ec2 (elastic compute service)  服务器

s3 (simple storage service) 静态资源存储 (对象, 文件, 静态页面...)
cloudFront

elb (elastic load balance) 负载均衡器

    auto scaling (弹性伸缩), 需要通过 cloud watch 设置好监视的告警阈值

ebs (elastic block store) 数据卷/硬盘, 分离硬盘数据存储和计算实例

rds (relational database service)
```
