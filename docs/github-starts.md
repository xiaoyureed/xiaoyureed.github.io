# 检索搜索技巧

```
# 项目名称中包含 xxx
in:name xxx

# start 数必须大于 1500, fork数必须大于 100, 语言必须为 java, 最后更新时间必须大于 九月三号
in:name xxx starts:>1500 forks:>100 language:java pushed:>2019-09-03

# 在 readme 中搜索
in:readme xxx

# 在 描述中搜索 (试用于用中文搜索)
in:description xxx


```
